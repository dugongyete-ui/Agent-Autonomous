## üîë CARA PAKAI MODEL AI (Step by Step)

### **Langkah 1: Daftar Hugging Face (GRATIS)**
1. Buka browser, pergi ke: **https://huggingface.co/join**
2. Klik **"Sign Up"**
3. Isi:
   - Email: email Anda
   - Username: nama pengguna (contoh: "akusaya123")
   - Password: buat password
4. Klik **"Create Account"**
5. Cek email, klik link verifikasi

### **Langkah 2: Dapatkan API Key**
1. Setelah login, klik foto profil (kanan atas)
2. Pilih **"Settings"**
3. Di sidebar kiri, klik **"Access Tokens"**
4. Klik tombol **"New token"**
5. Isi form:
   - Name: `chatbot-saya` (bebas)
   - Role: **"Read"**
   - Expiration: **"No expiration"**
6. Klik **"Create token"**
7. **COPY TOKEN** (penting! simpan baik-baik)

### **Langkah 3: Pilih Model**
Untuk pemula, saya sarankan:
```javascript
// Paling cepat & cocok untuk pemula

// Balance antara kualitas & kecepatan

// Kualitas terbaik (tapi lebih lambat)
```

## üß™ CONTOH TESTING

### **Test Script untuk Cek Model**
Buat file `test-model.js`:
```javascript
const { HfInference } = require('@huggingface/inference');
const hf = new HfInference('YOUR_API_KEY');

async function testModel() {
    try {
        const response = await hf.chatCompletion({
            model: "Qwen/Qwen2.5-1.5B-Instruct",
            messages: [{ role: "user", content: "Halo, siapa kamu?" }],
            max_tokens: 100
        });
        
        console.log("Response:", response.choices[0].message.content);
        console.log("Model berhasil digunakan!");
    } catch (error) {
        console.error("Error:", error.message);
    }
}

testModel();
```

## ‚ö†Ô∏è ERROR HANDLING (Yang Sering Terjadi)

### **Error: "Invalid API key"**
**Solusi:** 
- Pastikan API key benar (dimulai dengan `hf_`)
- Token harus dengan role "Read"
- Coba generate token baru

### **Error: "Model not found"**
**Solusi:**
- Cek penulisan nama model (case-sensitive)
- Gunakan nama lengkap: `Qwen/Qwen2.5-1.5B-Instruct`

### **Error: "Rate limit exceeded"**
**Solusi:**
- Tunggu 24 jam untuk reset limit
- Atau buat akun baru dengan email berbeda
