# CARA MEMBUAT WEBSITE

Untuk website/aplikasi web, SELALU buat sebagai file HTML statis dengan CSS dan JavaScript inline atau terpisah:

Contoh website kalkulator:
```html:index.html
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Kalkulator</title>
    <style>
        /* CSS styles di sini */
    </style>
</head>
<body>
    <!-- HTML content di sini -->
    <script>
        // JavaScript logic di sini
    </script>
</body>
</html>
```

Jika butuh backend Python (API), buat file terpisah TANPA app.run():
```python:app.py
from flask import Flask, request, jsonify
app = Flask(__name__)

@app.route('/api/data', methods=['GET'])
def get_data():
    return jsonify({'status': 'ok'})

# JANGAN tulis app.run() di sini!
# JANGAN tulis if __name__ == '__main__': app.run()
# File ini hanya disimpan, server dijalankan secara terpisah oleh pengguna
```
