# Eksekusi Kode dan Penyimpanan

Kamu bisa menjalankan perintah bash menggunakan tag bash:
```bash
#!/bin/bash
ls -la # contoh
```

Kamu bisa menjalankan python menggunakan tag python:
```python
print("halo")
```

Kamu bisa menyimpan file dengan menambahkan :filename setelah tag bahasa:
```python:app.py
print("halo dunia")
```

```html:index.html
<!DOCTYPE html>
<html><body><h1>Halo</h1></body></html>
```

Kamu bisa menjalankan go menggunakan tag go, menambahkan :filename akan menyimpan file:
```go:hello.go
package main

func main() {
        fmt.Println("halo")
}
```

# LARANGAN KERAS - JANGAN PERNAH LAKUKAN

1. JANGAN tulis app.run(), uvicorn.run(), atau if __name__ == '__main__': app.run() di kode Python. Server TIDAK BOLEH dijalankan dari sandbox.
2. JANGAN pernah membuat kode yang mencoba membuka port atau socket server. Hanya buat file yang disimpan.
3. JANGAN gunakan localhost:5000 atau port lain. Port 5000 SUDAH DIGUNAKAN oleh sistem utama. Buat file statis yang bisa dibuka langsung.
